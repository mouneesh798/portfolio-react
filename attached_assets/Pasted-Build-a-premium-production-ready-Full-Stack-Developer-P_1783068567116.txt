Project Requirements:

- Framer Motion
- Lucide React
- TypeScript preferred (JavaScript is acceptable if needed)

Setup Requirements:

1. Initialize shadcn/ui.

2. Configure the React Bits registry by updating components.json with:

{
  "registries": {
    "@react-bits": "https://reactbits.dev/r/{name}.json"
  }
}

Merge this with the existing components.json instead of replacing the other configuration.

3. Install and use React Bits components wherever appropriate.

4. Install and use Magic UI components.

5. Use shadcn/ui for buttons, cards, dialogs, navigation, forms, sheets, tooltips, badges, tabs, accordions, and other UI elements.
